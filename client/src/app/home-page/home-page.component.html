Client Manager Demo

Thank you for reviewing this application to the Multicert development software challenge.

In order to create clients, please select the "Create Client" option in the section of the navigation bar.

To view or delete clients, select the type of data display in the appropriate section of "List Clients". "Search Clients" allows one to look for clients based on the NIF or name of the client, and "All Clients" displays all clients. Both allow the deletion of a client through a button.
